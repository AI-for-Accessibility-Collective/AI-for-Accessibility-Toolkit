One-time setup needed. Custom skills run through Chrome's user-scripts feature, which needs two toggles turned on:
  1. Open chrome://extensions in a new tab.
  2. Turn on Developer mode in the top-right.
  3. Find Agentic A11y, click Details, and turn on Allow user scripts. (This toggle is per-extension and off by default — Chrome 138+ requires it separately from Developer mode.)
  4. Reload the extension at chrome://extensions.
